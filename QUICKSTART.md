
## üêõ Common Issues

**Images not showing?**
- Make sure the file is named exactly `profile.jpg`
- Check it's in the `images/` folder
- Try refreshing the page (Ctrl+F5)

**Animations not working?**
- Check browser console for errors (F12)
- Make sure jQuery is loading (check internet connection)

**Mobile menu not opening?**
- Clear browser cache
- Make sure JavaScript is enabled

## üöÄ Deploy Your Portfolio

### GitHub Pages (Free!)
1. Create a GitHub repository
2. Push your code
3. Go to Settings ‚Üí Pages
4. Select main branch
5. Your site will be live at `username.github.io/repository-name`

### Netlify (Free!)
1. Drag and drop your folder to [netlify.com/drop](https://app.netlify.com/drop)
2. Done! You get a free URL

### Vercel (Free!)
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in your project folder
3. Follow the prompts
